# My Project

> Built with [Le Socle](https://github.com/le-socle/socle) — a human-first method for working with AI agents.

---

## Get started

This repo comes with Le Socle pre-installed in `.socle/`.

### Step 1 — Open your AI tool

Open this repo in Claude Code, Codex, Cursor, or any AI coding tool.

### Step 2 — Say this

> **"Help me configure Le Socle for this project."**

The AI will read the briefing, understand the method, and help you fill in your manifest.

### Step 3 — Start coding

Your AI now understands your project, follows your procedures, and remembers what it learns.

---

## What's inside

```
.socle/
├── SOCLE.md              <- AI briefing (read once)
├── manifest.md           <- your project's constitution (fill this first)
├── memory/
│   ├── MEMORY.md         <- memory index
│   └── cortex/           <- specialized zones (fill as you work)
├── skills/               <- 7 operational skills
├── rules/                <- quality criteria
├── issue-board/          <- Kanban board (folder = status)
├── templates/            <- sprint template
└── scripts/              <- automation (generate-board.py)
```
